# Claude Code

## 常用指令
Enter 直接执行，Tab 可以补充内容，Esc 中断对话
1. 清除上下文：/clear
2. 压测上下文：/compact
3. 设置中文：/config language
4. 查看上下文使用量：/context 
5. 复制最后一次回复：/copy 
6. 导出对话：/export
7. 初始化 CLAUDE.md：/init，/memory 查看
8. 切换模型：/model
9. 进入计划模式：/plan
10. 插件：/plugin
11. 重命名当前对话名字：/rename
12. 对话切换：/resume
13. cc 状态查询: /status 
14. 配置状态栏：/statusline
